* **Sistem Setoran & Poin**: Perhitungan poin otomatis berdasarkan jenis dan berat sampah untuk mendorong budaya daur ulang.
* **Sticky Footer & Responsive Design**: Antarmuka yang rapi dengan teknologi Flexbox agar footer tetap berada di dasar layar, serta optimal untuk penggunaan mobile maupun desktop.
